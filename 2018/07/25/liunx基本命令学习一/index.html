linux基本命令学习一

Jul 25, 2018

基本的bash shell命令

##文件和目录列表

  • 显示当前目录下的文件和目录– ls
  • 区分文件和目录– ls -F
  • 显示隐藏文件– ls -a
  • 列出当前目录下包含的子目录中的文件，又名递归选项– ls -F -R
  • 显示文件附加信息–ls -l

    -l会产生长列表格式的输出，包含了目录中每个文件的更多相关信息，第一行显示了在目录中包含的总块数，每行都包含下述信息：

  • 文件目录，比如目录(d)、文件(-)、字符型文件(c)或块设备(b)
  • 文件的权限
  • 文件的硬链接总数
  • 文件属主的用户名
  • 文件属组的组名
  • 文件的大小(以字节为单位)
  • 文件的上次修改时间

  • 文件名或目录名

  • 当指定特定文件的名称作为过滤器时，ls命令只会显示该文件的信息。问号(?)代表一个字符；星号(*)代表零个或多个字符

##处理文件

  • 创建空文件，创建你指定的文件，并将你的用户名作为文件的属主，文件大小为零– touch

  • 将文件或目录从一个位置复制到另一个位置，–cp source destination源对象和目标对象

    1.若目标文件已存在，加上-i选项，强制询问是否需要覆盖已有文件
    2.递归的复制整个目录的内容，加上-R选项
    3.cp命令中也可使用通配符。制表键Tab自动补全可以允许你在输入文件名或目录名时按下制表键补全。连续按下制表键将展示所有相同开头的文件名

  • 为文件创建符号链接（一个文件指向存放在虚拟目录结构中某个地方的另一个文件，这两个通过符号链接在一起的文件，彼此的内容并不相同），原始文件必须存在– ln -s

  • 创建硬链接，原始文件也必须存在，带有硬链接的文件共享inode编号，因为它们是同一个文件– ln
  • 将文件和目录移动到另一个位置或重新命名– mv source destination
  • 删除文件，加上-i命令参数提示你是不是要真的删除文件。要删除很多文件且不受提示符的打扰，可以加上-f选项强制删除 –rm

##处理目录

  • 创建目录；同时创建多个目录和子目录，需要加上-p选项– mkdir directoryname
  • 删除目录；仅删除空目录– rmdir

##查看文件内容

  • 查看文件的类型– file
  • 显示文本文件的所有数据；给所有的行加上行数，加上-n选项；给有文本的行加上行号，加上-b选项；不想让制表符出现，加上-T选项– cat
  • 显示文本文件的内容，每页数据之后停下来– more
  • 显示文本文件的内容，实现文本文件的前后翻动，还有一些高级搜索功能– less
  • 显示文件最后几行的内容，默认情况下，显示文件末尾10行；通过加入-n参数来修改所显示的行数；在其它进程使用该文件时查看文件内容，加上-f选项– tail
  • 显示文件开头那些行的内容。默认情况下，显示文件前10行；通过加入-n参数来修改所显示的行数；– head
